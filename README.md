# md2pt_jp

Markdownで書かれた日本語小説を、小説投稿サイト（例：「小説家になろう」など）向けのプレーンテキストに変換するツールです。

## ✅ 対応している変換

- 見出し：
  - `###見出し3` → `【見出し3】`
  - `##見出し2` → `★★見出し2★★`
  - `#見出し1` → `☆EPISODE☆見出し1`
- 強調（`**強調**`）→ 強調マークを削除
- 箇条書き → `・` に置換
- 区切り線 `---` → `＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝`
- その他細かいMarkdown記法の除去

## 🚀 使い方

```bash
poetry run python -m md2pt_jp.cli input.md output.txt
